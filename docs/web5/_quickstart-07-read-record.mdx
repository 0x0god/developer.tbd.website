## 7. Reading messages in your DWN

We put the data in; let's get it out. All of this is from your 
personal datastore, your DWN, running embedded in your application.

```js
let readResult = await web5.dwn.records.read(did.id, {
    author: did.id,
    message: {
        recordId: queryResult.entries[0].recordId,
    },
});
```
