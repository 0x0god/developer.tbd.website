# Query from DWN

Before reading from your Decentralized Web Node (DWN), be sure to check out our [Quickstart guide](../../quickstart) to ensure you’ve properly imported the Web5 SDK. Additionally, make sure you’ve set up a DID before attempting to query from a DWN.

## Querying Protocols

The following snippet allows you to query for protocols within your `Web5` instance’s DWN:

```jsx
const { protocols } = await web5.dwn.protocols.query({
  message: {
    filter: {
      protocol: "https://music.org/protocol",
    },
  },
});

// logs an array of protocol configurations installed on the user's DWN
console.log(protocols);
```

## Querying Records

```js
const { records } = await web5.dwn.records.query({
  from: 'did:example:alice',
  message: {
    filter: {
      schema: 'https://schema.org/Playlist',
      dataFormat: 'application/json'
    }
  }
});

/// logs an array of records found on the user's DWN
console.log(records);
```

## Query Sorting

### Records

```js
// Sorting records by datePublished in ascending order
const response = await web5.dwn.records.query({
  message: {
    filter: {
      dataFormat: 'text/plain',
    },
    //highlight-start
    dateSort: 'publishedAscending'
    //highlight-end
  }
});
```

### Protocols

```js
// Sorting protocols by dateCreated in descending order
const { protocols } = await web5.dwn.protocols.query({
  message: {
    filter: {
      protocol: 'http://social-mediasss.xyz',
    },
    //highlight-start
    dateSort: 'createdDescending'
    //highlight-end
  }
});
```
